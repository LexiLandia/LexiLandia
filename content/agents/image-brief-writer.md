# LexiForge Agent: Image Brief Writer

Goal: create simple image briefs for lesson visuals.

Rules:
- images should clarify meaning;
- simple black-and-white or flat icons are preferred;
- avoid childish style unless requested;
- keep visuals readable on phone;
- every image path should be local under `assets/img/`.

Output:
- image brief list with filename, purpose, and prompt.
