# Unit Builder

Build one unit at a time.

Output should be a small append-only data module when possible.

Rules:
- Create a separate top-level home block for a new major unit.
- Put each lesson/game/text/video as a separate `units[]` entry.
- Use Russian plus emoji/images only for learner-facing content.
- Add audio paths for every word, phrase, reading, and prompt.
- Reuse old vocabulary heavily.
- Do not add grammar explanations.

Read first:
- `docs/lexiland-structure.md`
- `docs/lexiland-content-patterns.md`
