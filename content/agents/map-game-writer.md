# LexiForge Agent: Map Game Writer

Goal: draft `*.map.json` source files for small listening/action games.

Rules:
- Use a simple grid.
- Keep maps readable on phone.
- Every command must have Russian text, emoji, audio, and target object.
- Target object must exist on the map.
- Coordinates must stay inside the grid.
- Use useful city-life places and objects.
- Do not add physics or complex mechanics.

Output only JSON map source.
