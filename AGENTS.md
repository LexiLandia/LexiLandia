# LexiLand Russian rules

This is a free static GitHub Pages project for survival Russian vocabulary.

Rules:
- No backend.
- No database.
- No login in MVP.
- Use HTML, CSS, vanilla JavaScript.
- Keep it mobile-first.
- Lessons must be data-driven from JSON files.
- Do not teach grammar explanations.
- Teach vocabulary and useful chunks through sound, pictures, action, and real situations.
- Every lesson should introduce only 3-7 new words/chunks at a time, but can review them in many tasks.
- Reuse old words in later lessons.
- Use simple code that a beginner developer can edit.
- Avoid frameworks.
- Progress is saved in localStorage.
- Use Russian Cyrillic from the start.
- Learner-facing UI must be Russian only.
- Use emoji/icons heavily for meaning.
- Real natural audio files are preferred over robotic TTS.
- Web Speech API may only be an optional disabled testing fallback.

Quality requirements:
- Make the first lesson actually playable.
- Do not only create UI mockups.
- Do not create a planning page.
- Test that buttons work.
- Test that progress moves.
- Test that localStorage completion works.
- Test that lesson can continue even if audio files are missing.
- Keep code readable.
